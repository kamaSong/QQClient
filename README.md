课程设计项目 | Java + Socket + 多线程

## 项目简介

基于 Socket 的即时通讯系统客户端，与服务端建立 TCP 长连接，支持用户登录、私聊、群发、文件传输、在线用户列表等功能。

**开发周期**：2022.11 - 2022.12

## 技术栈

- Java
- Socket（TCP）
- 多线程
- 对象序列化

## 核心功能

- **用户登录**：发送用户名密码到服务端验证
- **私聊消息**：指定在线用户发送消息
- **群发消息**：向所有在线用户广播消息
- **在线用户列表**：请求并显示当前在线用户
- **文件传输**：读取本地文件，封装后发送
- **服务端推送日志**：接收并显示服务端推送的系统日志
- **控制台交互**：封装工具类处理输入校验

## 项目结构

| 文件 | 作用 |
|------|------|
| QQView.java | 主界面，菜单显示与交互 |
| UserClientService.java | 用户服务（登录/私聊/群发/退出） |
| FileClientServer.java | 文件传输服务 |
| ClientContentServerThread.java | 通信线程，监听服务端消息 |
| ManagerClientConnectThread.java | 线程管理（HashMap） |
| Utility.java | 控制台输入工具类 |
